# Student Grade Analyzer

Dự án Java cơ bản để phân tích điểm số học sinh, có áp dụng Unit Test.

## Chức năng chính
1. **countExcellentStudents**: Đếm số học sinh có điểm >= 8.0 (Bỏ qua điểm < 0 hoặc > 10).
2. **calculateValidAverage**: Tính điểm trung bình cộng của các điểm hợp lệ (0-10).

## Yêu cầu hệ thống
- Java JDK 8 trở lên.
- JUnit 5.

## Cách chạy kiểm thử
- Mở project bằng IntelliJ IDEA hoặc Eclipse.
- Vào thư mục `test`, chuột phải file `StudentAnalyzerTest.java` và chọn **Run**.
